ABSTRACT

This document presents the specifications of a suite of tests covering the operations of the MTOSI Rel 1.1 service interfaces.

It is expected to be used by product vendors willing to certify their MTOSI API implementation.

The tests focus on the semantic behavior of the operations complementing the Reference Implementation which is useful for the syntactic control of the API.

Many test specifications are complemented by XML files elaborated from a real network lab experiment. Those XML files representing request and response messages can be used in relation with the Reference Implementation.
