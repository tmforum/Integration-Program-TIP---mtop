Introduction

Purpose and Intended Audience

This deliverable elaborates a suite of test specifications for the different Services of the MTOSI Rel 1.1 release.
It is one of the elements supporting the conformance process.

It is expected to be used by product vendors willing to certify their MTOSI API implementation. Equipment vendors will typically implement the supplier side of MTOSI Services while OS software vendors will implement the requestor side.

Structure of the document

Section 2 presents the methodology that has been used to elaborate the test suite. It shows which structure has been chosen to organize the tests; it presents the test templates and the example network which has been used as support to illustrate some tests.

Section 3 presents the test scenarios to be considered as preconditions when interpreting the tests cases and examples.

Section 4 presents the test suite itself. There is one subsection per MTOSI Service.
