Class TcaParameterProfile

This object class contains per layer rate a set of PM parameter/threshold value pairs per granularity (e.g. VC12: ES / 50; 15min).
Instances of this class could be associated to TPs. A list of all associated TPs will be maintained in the profile.
Traceability: R_TMF518_NRA_I_0011 and R_TMF518_NRA_I_0012.
