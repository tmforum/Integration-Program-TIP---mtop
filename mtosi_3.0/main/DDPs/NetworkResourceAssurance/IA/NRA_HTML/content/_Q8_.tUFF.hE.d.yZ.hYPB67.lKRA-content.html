Class ProtectionGroup

This object class represents the information about a trail protection in a Managed Element.
Traceability: R_TMF518_NRA_I_0009 and R_TMF518_NRA_I_0010.

Attributes
ProtectionGroupType  type

This attribute identifies the type of the Protection Group.
ProtectionSchemeState  protectionSchemeState

This attribute identifies the state in which the protection scheme is in. FORCED_OR_LOCKED_OUT indicates that the entire group is locked; partial locking is indicated by AUTOMATIC.
Individual locks can be reported through transmission parameters on the appropriate TPs.
ReversionMode  reversionMode

This attribute identifies whether the switch is revertive or not.
 layerRate

This attribute identifies the line rate of the TPs of the Protection Group.
 parameterList

This attribute identifies any protection group specific parameters. Non-applicable parameters and parameters for which the value is unknown to the target OS may be left out.
See attached SD1-29_PGPParameters supporting document for values and descriptions.
String  apsProtocolType

This attribute defines the APS protocol on which the switching function is based on.

Possible values:
- "G.783" in case of K1/K2 based APS
- "Legacy" in case of proprietary APS protocol, as for radio equipment.
 asapRef

This attribute indicates the assignment of an Alarm Severity Assignement Profile (ASAP) to the Protection Group.
ItuArcStateAndStatusList  ituArcStateAndStatusList

Traceability: R_TMF518_NRB_I_0001 and R_TMF518_NRB_I_0004.
See supporting document SD1-8_encodingX731M3100.
 protectionRelatedTpRefList

This attribute contains the list of TPs that belong in the Protection Group. The list of TPs is partially ordered. The protecting TP always trails its worker TPs. The East TPs are always contiguous in the list, as well as West TPs.
