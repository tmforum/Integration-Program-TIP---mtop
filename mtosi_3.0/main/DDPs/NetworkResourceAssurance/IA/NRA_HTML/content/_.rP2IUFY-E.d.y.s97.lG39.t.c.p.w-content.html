Enumeration AcknowledgeIndication

This data type identifies if the alarm has been acknowledged or not.

Enumeration Literals
 ACKNOWLEDGED

This value is used in case that the Alarm has been acknowledged in the target OS. All Alarm fields other than osTime and acknowledge indication shall remain similar to the original alarm notification.
(The osTime is always provided as the time that the alarm acknowledgement notification has been reported by the target OS.)
 NOT_APPLICABLE

This value is used in case the target OS does not support acknowledgement for this event or does not support acknowledgement at all.
 UNACKNOWLEDGED

This value is used in case the event has not been acknowledged but the target OS supports acknowledgement for this event , or in case that the Alarm has been previously acknowledged and then unacknowledged. All Alarm fields other than osTime shall remain in that case similar to the original alarm notification.
(The osTime is always provided as the time that the alarm acknowledgement notification has been reported by the target OS.)
