Class EquipmentProtectionSwitchNotification

This object class represents the information in an equipment protection switch notification.

Attributes
EquipmentProtectionType  protectionType

This attribute identifies the type of the equipment protection.
EquipmentSwitchReason  switchReason

This attribute identifies the reason for the switch.
 epgRef

This attribute identifies the name of the Equipment Protection Group emitting the switch.
 protectedEquipmentRef

This attribute identifies the Equipment that was protected when the switch occurred. For a M:N group, protectedEquipmentRef always identifies the worker equipment instance for which the switch occured.
 switchAwayFromEquipmentRef

This attribute identifies the Equipment being switched away from.
 switchToEquipmentRef

This attribute identifies the Equipment which is being switched to.
 osTime

This attribute represents the time at which the event occurred at the reporting OS.
