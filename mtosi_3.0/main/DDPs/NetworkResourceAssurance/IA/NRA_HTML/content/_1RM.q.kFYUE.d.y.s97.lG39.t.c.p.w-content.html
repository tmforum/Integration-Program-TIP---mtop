Enumeration PerformanceMonitoringLocation

This enumeration identifies the location of the PM measure.
See attached supporting document LocationIdentification for an explanation.

Enumeration Literals
 BIDIRECTIONAL

Identifies a bidirectional measurement (i.e. both directions of the signal are taken into account for this measure).
 CONTRA_FAR_END_RX

Identifies a unidirectional measurement provided by a sink monitoring function that supervises the signal which is related to the source atomic function of this TP. The opposite direction of the supervised signal is measured.
See attached document for clarification.
 CONTRA_NEAR_END_RX

Identifies a unidirectional measurement provided by a sink monitoring function that supervises the signal which is related to the source atomic function of this TP. The direction of the supervised signal is measured.
See attached document for clarification.
 FAR_END_RX

See attached document for clarification.
 FAR_END_TX

 NEAR_END_RX

 NEAR_END_TX
