Type Definitions
Data Type ThresholdCrossingAlertParameter

This data type contains the TCA parameters assigned to the TCA Parameter Profile.
Traceability: R_TMF518_NRA_I_0013 and R_TMF518_NRA_I_0014.

Attributes
String  pmParameterName

This attribute holds the name of Performance Measurements. It has been defined as a string to accommodate proprietary extensions.
See attached SD1-28_PerformanceParameters supporting document for the list of defined Performance Measurements.
Granularity  granularity

This attribute defines the granularity of the TCA Parameter interval.
For counters two values "15_MIN" or "24_H" are supported. For current instantaneous measurements the granularity does not apply which is indicated by the value "NOT_APPLICABLE".
PerformanceMonitoringLocation  pmLocation

This attribute identifies the location of the PM Measurement.
See attached supporting document SD1-19_LocationIdentification.
PerformanceMonitoringThresholdType  thresholdType

This attribute identifies the type of the threshold value.
Trigger  trigger

This attribute indicates if the threshold is for the raise or the clear of a Threshold Crossing Alert.
 thresholdValue

This attribute identifies the value for the PM threshold parameter (real used to incorporate gauge PM parameters).
String  thresholdUnit

This attribute identifies the unit for the threshold value in a free format string.
