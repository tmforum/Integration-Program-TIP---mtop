Class TCANotification

This object class holds details of a Threshold Crossing Alert (TCA).
Traceability: R_TMF518_NRA_I_0016.

Attributes
Boolean  isEdgePointRelated

This attribute indicates whether the event is related to a Termination Point (TP) at the edge of a subnetwork.
It is True for edge points and their contained CTPs although it is always False as a CTP's attribute if the CTP doesn't terminate a Topological Link.
Note that the value is True for all notifications reporting a value change of the TP's attribute 'isEdgePoint' even if the new value is False.
 aliasNameList

This attribute contains a list of zero or more alias names for the object whose threshold has been crossed.
Boolean  isClearable

This attribute is True if the alarm/event represents a condition that will not be cleared at a later time (or is itself a clear); False otherwise.
Traceability: R_TMF518_NRA_I_0019.
 perceivedSeverity

This attribute identifies the severity of the TCA.
 layerRate

This attribute identifies the layer rate to which the TCA applies.
Granularity  granularity

This attribute defines the granularity of the TCA. For current instantaneous measurements the granularity does not apply which is indicated by the value "NOT_APPLICABLE".
String  pmParameterName

This attribute identifies the name of the Performance Measurements. It has been defined as a string to accommodate proprietary extensions.
PerformanceMonitoringLocation  pmLocation

This attribute identifies the location of the PM measurement.
PerformanceMonitoringThresholdType  thresholdType

This attribute identifies the type of the threshold which has been crossed.
 thresholdValue

This attribute identifies the details of the threshold that has been crossed (real used to incorporate gauge PM parameters).
String  thresholdUnit

This attribute identifies the unit of the measurement of the threshold that has been crossed in a free format string.
AcknowledgeIndication  acknowledgeIndication

This attribute identifies if the alarm has been acknowledged or not. In case the target OS does not support acknowledgement for this event or does not support acknowledgement at all NOT_APPLICABLE shall be used.
Traceability: R_TMF518_NRA_I_0025.
