Type Definitions
Data Type PerformanceMonitoringThreshold

This data type represents a Performance Monitoring Threshold. It covers the ability of the NE to supervise thresholds on a specific PMParameter (i.e. if the NE is able to supervise thresholds on a PM parameter the data type PMThreshold has to exist, otherwise will never exist).
Traceability: R_TMF518_NRA_I_0006.

Attributes

This attribute identifies the type of the threshold value.
Trigger  trigger

This attribute indicates if the threshold is for the raise or the clear of a Threshold Crossing Alert.
 thresholdValue

This attribute identifies the value for the PM threshold parameter (real used to incorporate gauge PM parameters).
String  thresholdUnit

This attribute identifies the unit for the threshold value in a free format string.
