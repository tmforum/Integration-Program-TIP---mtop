Type Definitions
Data Type PerformanceMonitoringParameter

This data type lists the PM parameter supported by the PMP, e.g. PMP_ES, PMP_SES, etc. together with all their thresholds.
Traceability: R_TMF518_NRA_I_0005.

Attributes
String  pmParameterName

This attribute holds the name of Performance Measurements. It has been defined as a string to accommodate proprietary extensions.
See attached SD1-28_PerformanceParameters supporting document for the list of defined Performance Measurements.
PerformanceMonitoringThreshold  pmThresholdList

This attribute lists all configured thresholds of the PM parameter.
