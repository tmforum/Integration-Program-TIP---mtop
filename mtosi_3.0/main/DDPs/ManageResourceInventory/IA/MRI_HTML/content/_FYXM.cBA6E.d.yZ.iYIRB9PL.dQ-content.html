 getAllManagedElements

This operation returns all the Managed Elements contained by a provided Management Domain or Multi Layer Subnetwork.

Traceability: R_TMF518_MRI_II_0015, R_TMF518_MRI_II_0017.
 getAllManagedElementsPassingFilter

This operation returns all Managed Elements (MEs) contained by a specific Management Domain (MD) that pass a given filter condition.
The filter condition should test for equality based on any combination of location, manufacturer, productName and resourceState. The matching is based in "AND", i.e., all the supplied filter conditions must be met. If the filter is left empty, then all the MEs contained by the given MD are returned.
The operation can only be directed to a top-level OS and not to a subordinate OS.

Traceability: R_TMF518_MRI_II_0001.
 getAllManagedElemetsWrtOs

This operation returns all Managed Elements (MEs) managed by the provided subordinate OS AND contained by the provided Management Domain (MD).
If no MD is provided all MEs managed by the provided subordinate OS are returned.
Note that "Wrt" in the name of the operation stands for "with respect to".

Traceability: R_TMF518_MRI_II_0007, R_TMF518_MRI_II_0013, UC_TMF518_MRI_0012.

 getAllManagedElementsPassingFilter 
Public  getAllManagedElementsPassingFilter(ManagedElementFilter, ,  )

This operation returns all Managed Elements (MEs) contained by a specific Management Domain (MD) that pass a given filter condition.
The filter condition should test for equality based on any combination of location, manufacturer, productName and resourceState. The matching is based in "AND", i.e., all the supplied filter conditions must be met. If the filter is left empty, then all the MEs contained by the given MD are returned.
The operation can only be directed to a top-level OS and not to a subordinate OS.

Traceability: R_TMF518_MRI_II_0001.
