Type Definitions
Data Type RouteAndRouteType

This data type defines a list of Routing Area identifiers, the type of route with which they are associated (home/intended route, actual/current route or both) and a brief description.

Attributes
String  raIdList

This attribute contains the identifiers of the Routing Areas (RAs) participating in the route of a Connection.
RouteType  routeType

This attribute indicates the type of the route, home/intended or actual/current.
String  routeDescription

This attribute provides a description of the route.
Including the degree of detail available from the Control Plane (depending on the signalling), i.e. whether the route description is available in a "sparse" or "full" form:
- Full
The Control Plane component at the originating point of the Connection has a full view of all of the ids of the Routing Area one level subordinate to the top-level Routing Area involved in the Connection.
- Sparce
The Control Plane component at the originating point of the Connection can only provide the id of the local Routing Area, the next neighbour and the destination Routing Area (and can not provide the identifiers of any other Routing Areas along the route).
