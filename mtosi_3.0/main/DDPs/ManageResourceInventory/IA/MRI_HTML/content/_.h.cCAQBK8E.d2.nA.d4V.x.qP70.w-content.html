Type Definitions
Data Type IncludedObjectType

This data type defines a single object type and the associated kind of information to be included in the InventoryFilter data type.

Attributes
String  objectType

This attribute defines the object type to be returned.
The following abbreviations are used for the object type names:
MD - management domain
ME - managed element
EH - equipment holder
EQ - equipment
PTP - physical termination point
FTP - floating termination point
CTP - connection termination point
TPPool - termination point pool
EPGP - equipment protection group
PGP - protection group
SNC - subnetwork connection
RT - route
CC - cross connection
SN - subnetwork
TL - topological link
OS - operations system
TMD - transmission descriptor.
Include  include

This attribute indicates whether
a. just the name,
b. all the associated attributes (including the name), or
c. the associated attributes and relationship pointers
are to be returned for each instance.

If include is "NAME" then just the name of each object instance is returned, if include is "ATTRS" all the associated attributes (including the name) for each instance are returned, and if include is "FULL" then the associated attributes (including the name) and the relationship pointers are returned for each instance.
