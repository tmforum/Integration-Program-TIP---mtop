 getTransmissionParameters

This operation retrieves the layered Transmission Parameters of a
- Termination Point (TP)
- Matrix Flow Domain (MFD)
- Flow Domain (FD)
- Flow Domain Fragment (FDFr) or
- Transmission Descriptor (TMD)
given the name of the object and providing a filter that is based on the Layer Rates and
- Level 1 headings defined in the attached supporting document SD1-16_LayeredParameters
- Level 2 headings defined in the attached supporting document SD1-16_LayeredParameters
- explicitly listed, i.e., MSTI_Table, VID_to_MSTI_Table, CTagTranslation_Table, STagTranslation_Table and TrafficMapping_Table.

It is also possible to retrieve all layered Transmission Parameters defined in the Resource.
For a list of the currently defined set of supported Transmission Parameters refer also to the attached supporting document SD1-16_LayeredParameters.

Traceability: R_TMF518_MRI_II_0107, R_TMF518_MRI_II_0114, R_TMF518_MRI_II_0122, R_TMF518_MRI_II_0132, R_TMF518_MRI_II_0152.

This operation retrieves the layered Transmission Parameters of a
- Termination Point (TP)
- Matrix Flow Domain (MFD)
- Flow Domain (FD)
- Flow Domain Fragment (FDFr) or
- Transmission Descriptor (TMD)
given the name of the object and providing a filter that is based on the Layer Rates and
- Level 1 headings defined in the attached supporting document SD1-16_LayeredParameters
- Level 2 headings defined in the attached supporting document SD1-16_LayeredParameters
- explicitly listed, i.e., MSTI_Table, VID_to_MSTI_Table, CTagTranslation_Table, STagTranslation_Table and TrafficMapping_Table.

It is also possible to retrieve all layered Transmission Parameters defined in the Resource.
For a list of the currently defined set of supported Transmission Parameters refer also to the attached supporting document SD1-16_LayeredParameters.

Traceability: R_TMF518_MRI_II_0107, R_TMF518_MRI_II_0114, R_TMF518_MRI_II_0122, R_TMF518_MRI_II_0132, R_TMF518_MRI_II_0152.

Parameters:
layerRateList [*]This parameter provides the Layer Rates of the Transmission Parameters that have to be returned.
If no Layer Rate is provided, all Layer Rates supported by the Resource are in scope.
resourceRef [1]This parameter provides the name of the Resource whose Transmission Parameters are to be retrieved.
String transmissionParameterGroupFilterList [*]This parameter provides the Transmission Parameter Groups that have to be returned by providing
- Level 1 headings defined in the attached supporting document SD1-16_LayeredParameters
- Level 2 headings defined in the attached supporting document SD1-16_LayeredParameters
- explicitly listed parameter groups, i.e., MSTI_Table, VID_to_MSTI_Table, CTagTranslation_Table, STagTranslation_Table and TrafficMapping_Table.

If no Group is provided, all Transmission Parameters supported by the Resource are in scope.
transmissionParameterList [*]This parameter returns the requested Transmission Parameters of the provided Resource.
