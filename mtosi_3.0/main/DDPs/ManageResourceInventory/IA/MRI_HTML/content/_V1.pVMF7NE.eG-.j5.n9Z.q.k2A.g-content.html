Enumeration RouteType

This data type defines the type of a route.

Enumeration Literals
 ACTUAL_CURRENT

This value indicates that the route is the actual (current) route.
 BOTH

This value indicates that the route is both the home (intended) route and the actual (current) route.
 HOME_INTENDED

This value indicates that the route is the home (intended) route.
