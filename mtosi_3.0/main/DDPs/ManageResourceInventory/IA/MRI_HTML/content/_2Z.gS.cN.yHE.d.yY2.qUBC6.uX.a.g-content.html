 getConnectivityAwareness

This operation returns the supported connectivity awareness capability (with regard to the connectionless technologies) provided by the target OS.
The target OS works either in the “connectivity-aware” or in the “connectivity-unaware” mode:
- “connectivity aware” indicates that the target OS has some capability to detect the connectivity between the MFDs within an FD
- “connectivity-unaware” indicates that the target OS has no capability at all to detect connectivity between the MFDs within an FD.

Traceability R_TMF518_MRI_V_0154.
