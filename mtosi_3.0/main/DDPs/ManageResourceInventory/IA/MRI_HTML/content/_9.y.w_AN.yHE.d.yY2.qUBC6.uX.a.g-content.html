Interface MultiLayerSubnetworkRetrievalService

 getAllMultiLayerSubnetworksWrtOs

This operation returns all Multi Layer Subnetworks (MLSNs) managed by the provided subordinate OS AND contained by the provided Management Domain (MD).
If no MD is provided all MLSNs managed by the provided subordinate OS are returned.
Note that "Wrt" in the name of the operation stands for "with respect to".

Traceability: R_TMF518_MRI_II_0007, R_TMF518_MRI_II_0013, UC_TMF518_MRI_0012.

This operation returns all Multi Layer Subnetworks (MLSNs) managed by the provided subordinate OS AND contained by the provided Management Domain (MD).
