Type Definitions
Data Type TransmissionParameterList

This data structure contains a list of transmission parameters per layer rate. The list of defined parameters can be fond in the SD1-16_LayeredParameters supporting document.
Note: Setting is prohibited for some specific parameters.
Traceability: R_TMF518_NRB_I_0005.

Attributes
LayerRate  layerRate

This attribute represents the layer and bandwidth (rate) to which the transmission parameters apply.
Specifically, for a CTP, identifies the layer of the CTP, and the connection rate supported by the CTP.
For a PTP, identifies the layer rate (bandwidth) of the PTP and the characteristic information of the PTP.
For an ATM TP, identifies the layer (and not the bandwidth) of the ATM TP.
See attached supporting document SD1-17_LayerRates for the list of defined layer rates.
 transmissionParameterList

This attribute contains the transmission parameters for the resource.
See SD1-16_LayeredParameters supporting document for the list of defined transmission parameters.
