Type Definitions
Data Type MultiEventInventoryAttributes

This data type is used when an object instance is embedded in the inventory layout payload of a Multi-Event Inventory (MEI) notification.

Attributes
ItuTTime  neTime

This attribute represents the time at which the network element (NE) detected the inventory change.
String  eventIndication

This attribute represents an indication of the type of inventory update.
Possible values are Object Creation (OC), Object Discovery (ODsc), Object Deletion (OD), Attribute Value Change (AVC), State Change (SC) and AVC_SC_and_Children.
Only the acronym (e.g., OC) is used in the actual attribute value.
The following conventions apply:
- If an object creation or object discovery indicator is applied to a parent object, it implicitly applies to all the children, i.e., the OS sending the notification only needs to set the indicator attribute in the parent object.
- If an object deletion is applied to a parent object, it is assumed that all the children are also deleted. The childern should not be listed.
- The AVC_SC_and_Children indication may be applied to a parent object to imply that there may be AVC or SC changes to the parent and some or all of the children.
