Operation Details

 subscribe 
Public  subscribe(String, String, String, String )

This operation allows the requesting OS to subscribe for notifications.

Parameters:
String consumerEpr [1]This parameter contains the notification consumer endpoint (callback handler) that the broker will call to send the notification.
String selector [1]This parameter contains the optional "filter" expression to restrict the flow of messages sent to the notification consumer. By default, an empty selector defines the desire to be notified of all events for the given topic.
String subscriptionId [1]This parameter contains a unique subscription identifier that is returned to the requesting OS to be used when invoking the unsubscribe() for the given topic. Note that it must uniquely identify the request signature by its three elements; consumerEpr, topic, and selector.
String topic [1]This parameter contains the unique name of the topic object of the subscription.
