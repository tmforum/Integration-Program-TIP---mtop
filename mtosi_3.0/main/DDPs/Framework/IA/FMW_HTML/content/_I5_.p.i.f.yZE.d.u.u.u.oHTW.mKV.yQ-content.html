Class LogCapacityThresholdAlarmNotification

This notification is a special type of Threshold Crossing Alert (TCA) Notification used to indicate that the Log capacity alarm threshold attribute has been crossed. Object Lifecycle: Created by the reporting OS.
Traceability: R_TMF518_FMW_I_0013.

Attributes
Integer  observedValue

This attribute represents the current Log size, as a percentage of the maximum Log size.
Integer  crossedValue

This attribute represents the threshold level that has been crossed.
PerceivedSeverity  perceivedSeverity

This attribute represents the severity of the alarm. The perceived severity is minor if the Log is not full and critical otherwise.
