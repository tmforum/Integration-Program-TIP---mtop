This operation sets the native EMS Name of the specified object.

 setAdditionalInfo

This operation sets the additional info attribute of the object (i.e. resource).
As an input only the list of parameters to be changed, removed, or added shall be provided. If an entry is to be removed, "-" shall be specified as a value. If a parameter is specified that is currently not part of the additionalInfo attribute of the resource that parameter is added by the target OS with the specified value.
The operation is best effort.
The output specifies the values which were actually set.
