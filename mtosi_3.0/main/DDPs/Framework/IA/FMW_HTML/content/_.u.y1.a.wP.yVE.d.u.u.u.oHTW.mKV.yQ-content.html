Enumeration OperationStatus

This enumeration identifies the status of a Managed Element with respect to the current database backup operation.

Enumeration Literals
 ABORTED

Aborted is used when last backup operation failed.
 COMPLETED

Completed is used when last backup operation was successful.
 IDLE

Idle is used when no database backup operation has been performed since the OperationsSystem last (re)started (boot).
 IN_PROGRESS

InProgress is used when a backup operation is being performed.
