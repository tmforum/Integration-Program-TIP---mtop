Class LogAttributeValueChangeNotification

This notification is a special type of Attribute Value Change Notification used to indicate that the following Log attributes have changed: - capacity alarm threshold - log full action - maximum log size - start time - stop time - week mask - changing the discrimnator constraint - max record life. Object Lifecycle: Created by the reporting OS.
Traceability: R_TMF518_FMW_I_0013.
