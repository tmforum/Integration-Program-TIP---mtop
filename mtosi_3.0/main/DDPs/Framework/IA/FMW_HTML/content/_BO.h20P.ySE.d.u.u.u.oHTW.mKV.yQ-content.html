Class HeartbeatNotification

This Notification is used to indicate the state of the notification delivery mechanism between the sending and receiving OSs. The sending OS shall send out heartbeat notifications to the receiving OS on a regular basis so that the receiving OS can conclude that it is out of sync (regarding events) if it doesn't receive any heartbeat notifications for some time.
