Class FileTransferStatusNotification

This event holds details of PM historical PM data file being transferred including transfer status, percent complete and failure reason.
The number of events indicating InProgress that will be transferred is target OS dependent, however, at least one event indicating Completed with percentComplete = 100, or Failed with failureReason filled in is mandatory.
Traceability: R_TMF518_FMW_I_0012.

Attributes
String  fileName

This attribute represents the name of the file being transferred (this shall include the path name).
FileTransferStatus  transferStatus

This attribute represents the current state of the transfer (in progress, failed or completed).
Integer  percentComplete

This attribute indicates the percent complete of the file transfer. It must be in the range 0...100. Only applies if transferStatus is InProgress or Completed.
String  failureReason

This attribute represents, in the event of a failure, the reason for the failure. Only applies if transferStatus is Failed.
