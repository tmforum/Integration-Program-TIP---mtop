Class MultiEventNotifications

This notification reports changes of multiple entities.
Traceability: R_TMF518_FMW_II_0033 and R_TMF518_FMW_II_0036.

Attributes
String  notificationIdentifier

This parameter identifies the identifier of the multi event notification. The uniqueness and the sequence of the notificationIdentifier are not guaranteed.
ItuTTime  osTime

This attribute represents the time at which the sending OS generated the notification.
EventType  eventType

This parameter indicates the type of event being reported.
AttributeValuePair  additionalInfo

This attribute is a vendor specific attribute that is used by the OS supplier to append additional parameters to the MEI notification. This list can be empty.

This is a vendor specific attribute that contains additional specific information about the object. This list can be empty.
The attribute, when present, can be used to convey information from the target OS to the requesting OS, and vice versa, of additional information that isn't explicitly modelled, except that some parameter names and values may be predefined.
Any information encoded in the additionalInfo is optional: the target OS need not fill out this information and the requesting OS need not interpret this information.
