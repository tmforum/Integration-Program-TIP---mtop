Type Definitions
Data Type CommonObjectSetData

This parameter provides the attributes describing the Group Termination Point to be created.

Traceability: R_TMF518_RP_II_0078.

Attributes
String  userLabel

This parameter defines the provisionable, user friendly name for the new/modified object.
Boolean  forceUniqueness

This parameter defines if uniqueness of the provided userLabel is required amongst all objects of this type within the target OS domain. True = uniqueness required; False = uniqueness not required.
String  owner

This parameter defines the label of the owner of the new/modified object.
String  nativeEmsName

This parameter contains a native EMS name for the new/modified object.

This attribute contains a list of aliases for the entity.
AttributeValuePair  aliasNameList

This parameter contains a list of aliases for the new/modified object.

This attribute contains a list of aliases for the entity.
AttributeValuePair  additionalInfo

This Parameter allows the communication from the requesting OS to the target OS of additional information which is not explicitly modelled.
