Class ObjectDiscoveryNotification

An Object Discovery Notification (ODscN) is an event used to indicate that an object has been discovered but not yet named by the naming OS, in case the discovery and naming OS are different OSs.
For further explanation see TMF518_FMW (R_TMF518_FMW_I_0012).
Object Lifecycle: Created by the reporting OS.

Attributes
String  discoveredName

This attribute serves as an identifier for the object against which the event notification is generated. The naming OS for this object will in due course provide a unique name for this object.
ObjectType  objectType

This attribute identifies the type of the object associated with the event. The attribute is needed to allow simple notification filtering based on the object type.
ItuTTime  osTime

This attribute represents the time at which the event occurred at the reporting OS.
