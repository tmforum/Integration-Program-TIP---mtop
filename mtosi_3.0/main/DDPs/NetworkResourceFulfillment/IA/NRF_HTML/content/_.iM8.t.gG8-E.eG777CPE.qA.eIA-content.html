Enumeration ConnectionType

This enumeration defines the type of the Connection; i.e., how the Connection object was created.

Enumeration Literals
 PC

This value indicates that the Connection is a Permanent Connection (PC); a Connection that has been created via establishCall() where there is no Control Plane.
 SC

This value indicates that the Connection is a Switched Connection (SC); a Connection that has been created by the network, at UNI.
 SPC

This value indicates that the Connection is a Soft Permanent Connection (SPC); a Connection object that has been created either:
- as a side effect of establishCall(), or
- by the network, at E-NNI, as a side effect of establishCall() performed at another interface.
