Class FloatingTerminationPoint

This object class represents a TP without a physical layer that behaves both like a PTP and a CTP.
Floating Termination Points (FTPs) are contained in managed elements as PTPs.
Note: The value of tpMappingMode for FTPs considers exclusively mapping on the server side.
Traceability: R_TMF518_NRF_I_0011.
