Type Definitions
Data Type CrossConnect

This data type represents a physical connection within a Managed Element. That is, a Cross-Connect is identified in this interface by its A end(s) and Z end(s), its "SNC" type, and its directionality. Other characteristics of SNCs do not apply to Cross-Connections. In particular, the static protection level does not apply to Cross-Connections, as reliability of Cross-Connections is not directly impacted by equipment protection at the interface. As well, the SNC state does not apply to Cross-Connections as a Cross-Connect represents a physical connection and not a logical, potential, or reserved set of connections. As such, a Cross-Connection may only be "active" or "inactive". A "partial Cross-Connection" actually corresponds to a different Cross-Connection in this interface as it has a different external shape (e.g., a "half" bidirectional simple Cross-Connect is not modelled as a partial Cross-Connect, but rather as a "complete" unidirectional simple Cross-Connection).
Traceability: R_TMF518_NRF_I_0033 and R_TMF518_NRF_I_0034.

Attributes
Boolean  isActive

This attribute identifies the state of the Cross-Connection. It is True if the Cross-Connection is active in the Managed Element and False otherwise. An "active" Cross-Connection means that no further intervention on the Cross-Connection itself is required from the requesting OS to activate it. The fact that a Cross-Connection is "active" does not necessarily imply that there is a traffic flow on the network.
ConnectionDirection  direction

This attribute identifies the direction of the Cross-Connection.
SubnetworkConnectionType  crossConnectType

This attribute indicates the external traffic flow of the requested Cross-Connection through the Managed Element.
Boolean  isFixed

This attribute defines if the CrossConnect is fixed (i.e. cannot be deleted by the requesting OS) or is flexible.
String  connectionId

This attribute represents the control plane identifier. It corresponds to the Connection Name attribute in G.7713.
Note that also the Cross Connection points to its superior Connection.
