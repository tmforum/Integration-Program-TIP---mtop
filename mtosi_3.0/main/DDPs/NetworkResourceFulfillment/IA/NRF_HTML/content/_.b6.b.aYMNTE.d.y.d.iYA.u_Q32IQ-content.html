Class MultiLayerSubNetwork

This object class is an abstraction of the topology provided by the target OS which describes the potential for SubNetwork Connections. Top-level Multi Layer Subnetworks are subnetworks contained directly under either the target OS or a Management Domain.
Traceability: R_TMF518_NRF_I_0021 and R_TMF518_NRF_I_0021.
