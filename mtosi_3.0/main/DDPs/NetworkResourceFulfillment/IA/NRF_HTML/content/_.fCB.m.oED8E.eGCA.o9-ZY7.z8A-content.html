Class MultiLayerSubNetworkPointPool

This object class represents a collection of layered SNPPs. There is one layered SNPP per supported layer rate.
A MLSNPP is composed by SNPPs which represent TP functions which have a G.805 TCP/TerminationFunction (Termination Connection Point). Otherwise SNPPs forms part of an MLSNPPLink if they represent TP functions which have a G.805 CP (Connection Point) role.

Note that multi-layer does not imply client-server relationship, but flexible mapping at the same switching layer.
For more details see SD1-45_ASONControlPlaneManagement-Primer.

Traceability: R_TMF518_NRF_I_0094, R_TMF518_NRF_I_0095.

Attributes
Directionality  direction

This attribute represents the directionality of its SNP components. All SNPs within a MLSNPP shall have the same direction.
The direction of the SNP is inherited from the TP (CTP/PTP/FTP) that the SNP represents.
«mTOPClass» LayeredSubNetworkPointPool  layeredSnppList

This attribute contains the list of Layered SNPPs that comprise the MLSNPP. One Layered SNPP per supported layer rate.
