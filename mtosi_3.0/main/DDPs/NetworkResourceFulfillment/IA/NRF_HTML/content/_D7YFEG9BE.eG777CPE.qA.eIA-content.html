Enumeration ConnectionState

This enumeration defines the state values for a Connection object.

Traceability: R_TMF518_NRF_I_0084.

Enumeration Literals
 COMPLETE

This value indicates that ALL resources necessary to support the Connection have been successfully allocated.
 NOT_APPLICABLE

This value indicates that the Connection is used in a non Control Plane environment.
 SEARCHING

This value indicates that NOT all the resources necessary to support the Connection have been successfully allocated.
This state may not be applicable in a non Control Plane environment.
