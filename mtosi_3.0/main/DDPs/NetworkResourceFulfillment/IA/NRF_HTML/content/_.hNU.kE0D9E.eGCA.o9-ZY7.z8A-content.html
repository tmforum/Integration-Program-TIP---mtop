Class LayeredSubNetworkPointPool

This object class represents a layered Subnetwork Point Pool (SNPP). A LayeredSNPP consists of a layer rate and the SNPP objects at that layer.

Traceability: R_TMF518_NRF_I_0102.

Attributes
 layerRate

This attribute represents the layer rate of the SNPP.
«mTOPClass» SubNetworkPointPool  snppList

This relationship represents the SNPP objects across the levels of routing hierarchy. It shall always include the details for the SNPPs in the relevant layer directly represented by the MLSNPP and may also include details for the subordinate SNPPs if the information is available to the target OS. The routing level is identified via the Routing Area IDs.
