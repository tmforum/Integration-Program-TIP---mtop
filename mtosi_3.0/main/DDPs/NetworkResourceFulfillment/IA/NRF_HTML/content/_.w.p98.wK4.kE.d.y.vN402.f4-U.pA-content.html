Class FlowDomainFragment

This object class is a logical entity that represents a transparent end-to-end connectivity between two or more Flow Points.

Traceability: R_TMF518_NRF_I_0056.

Attributes
ConnectionDirection  direction

This attribute represents the directionality of the FDFr.
 transmissionParameterList

This attribute contains the technology-specific parameters associated with the different connectionless layers that are supported by the FDFr.
Refer to to chapter “Connectionless Technology Parameters” of the attached supporting document SD1-16_LayeredParameters for details of the currently defined Transmission Parameters.
TerminationPointData  aEndTpDataList

This attribute lists all A End Flow Point names together with their TP data (TP Mapping Mode, Transmission Parameters).
TerminationPointData  zEndTpDataList

This attribute lists all Z End Flow Point names together with their TP data (TP Mapping Mode, Transmission Parameters).
Boolean  isFlexible

This attribute indicates whether the FDFr is fixed (= True) or is flexible (= False).
 administrativeState

This attribute defines administratively the role that the Route shall play for the belonging SNC. There is not any relationship with the actualState attribute of the Route.
It can have the following values:
- "locked", the route is not allowed to be active
- "unlocked", the route is allowed to be active.

This attribute indicates whether the FDFr is locked (i.e., traffic units cannot flow through the FDFr) or unlocked (i.e., traffic units are allowed to flow through the FDFr).
SubnetworkConnectionState  state

This attribute indicates if the FDFr is active or partial.
FlowDomainFragmentType  type

This attribute indicates the type of the FDFr.
 ituArcStateAndStatusList

Traceability: R_TMF518_NRB_I_0001 and R_TMF518_NRB_I_0004.
See supporting document SD1-8_encodingX731M3100.
 asapRef

This attribute indicates the assignment of an Alarm Severity Assignement Profile (ASAP) to the Flow Domain Fragment.
