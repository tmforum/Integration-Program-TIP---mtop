Class SubnetworkPoint

This object class represents the Subnetwork Point (SNP) object. An SNP object is a Control Plane representation of a Management Plane resource (the CTP). It consists of an identifier, an alternative identifier (i.e. TNA) and the name of CTP that it represents.

Traceability: R_TMF518_NRF_I_0104.
