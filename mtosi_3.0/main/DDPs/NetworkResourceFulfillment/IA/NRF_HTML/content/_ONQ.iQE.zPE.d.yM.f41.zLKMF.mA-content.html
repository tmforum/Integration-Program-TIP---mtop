Type Definitions::RerouteAllowed
EnumerationLiteral NOT_APPLICABLE

This value is used when the requesting OS does not want to specify the exact target OS behaviour. In this case it is left up to the target OS to decide whether rerouting will be provided. It is also used if the target OSs/Managed Elements are allowed to reroute the SNC but not required to attempt to reroute it upon failure.
