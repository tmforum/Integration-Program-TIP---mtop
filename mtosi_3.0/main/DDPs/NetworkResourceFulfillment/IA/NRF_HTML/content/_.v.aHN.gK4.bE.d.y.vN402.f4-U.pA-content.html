Class MatrixFlowDomain

This object class is contained within a Managed Element (ME) and represents a grouping of assigned Connectionless Port Termination Points (CPTPs).
Traceability: R_TMF518_NRF_I_0047 and R_TMF518_NRF_I_0048.

Attributes
 transmissionParameterList

This attribute contains the technology-specific parameters associated with the different connectionless layers that are supported by the MFD.
Refer to to chapter “Connectionless Technology Parameters” of the attached supporting document SD1-16_LayeredParameters for details of the currently defined Transmission Parameters.
Boolean  isFlexible

This attribute indicates whether the MFD is fixed and cannot be configured by the NMS (= False) or is flexible (= True).
