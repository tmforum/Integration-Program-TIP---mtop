Data Type EquipmentHolderInventory

This data type contains the data structure of the Equipment Holder as defined in the inventory layout.

Attributes

This attribute contains the name of the Equipment Holder.

This attribute contains the inventory of the Equipment contained directly in this Equipment Holder.
