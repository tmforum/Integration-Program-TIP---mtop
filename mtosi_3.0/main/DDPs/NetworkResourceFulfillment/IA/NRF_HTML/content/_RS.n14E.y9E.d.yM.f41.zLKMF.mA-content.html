Enumeration Reroute

This enumeration indicates if the target OS/MEs/Control Plane are allowed and/or required to reroute this SNC if there is a failure on this SNC, periodically to optimize the routes, of for any other reason.
This can be done either using network routing protocols or through centralized management.

Enumeration Literals
 NO

The NO value means that the target OS/MEs/Control Plane are not allowed to reroute the SNC.
 NOT_SET

The NOT_SET value is used when the requesting OS does not want to specify the exact behaviour. In this case it is left up to the target OS/(MEs/Control Plane to decide whether rerouting will be provided. It is also used if the target OS/MEs/Control Plane are allowed to reroute the SNC but not required to attempt to reroute it upon failure.
 YES

The YES value means that the target OS/MEs/Control Plane are allowed to reroute the SNC and required to attempt to reroute it upon failure.
