Type Definitions::SubnetworkConnectionState
EnumerationLiteral PENDING

The SNC has been created by a requesting OS and has not been activated by any requesting OS; or the SNC has been successfully deactivated by a requesting OS. That state has no relationship with the network state of the cross- connects of the SNC. It is allowable for a target OS to not support the SNCS_PENDING state and to reject any operation that attempts to put an SNC into SNCS_PENDING state.
