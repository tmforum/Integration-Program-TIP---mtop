Class FlowDomain

This object class represents a network grouping of zero or more Matrix Flow Domains (MFDs) and zero or more Connectionless Port Termination Points (CPTPs) that are assigned to these MFDs.
Traceability: R_TMF518_NRF_I_0049 and R_TMF518_NRF_I_0050.
