Data Type InventoryData

This data type contains the inventory layout structure as described in the attached supporting document SD2-12_Inventory_Layout.

Attributes

This attribute contains the list of Management Domain inventories.

This attribute contains the list of top level Transmission Descriptor inventories.

This attribute contains the list of Operations System inventories.

This attribute contains the list of Vendor Objects.
