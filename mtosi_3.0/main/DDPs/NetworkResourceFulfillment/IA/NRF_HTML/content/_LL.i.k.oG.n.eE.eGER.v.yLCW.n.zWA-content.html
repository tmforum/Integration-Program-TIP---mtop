Enumeration LevelOfEffort

This enumeration defines the level of effort that should be used in statisfying a particular request.

This attribute indicates that the level of effort is best-effort.
 MANDATORY

This attribute indicates that the level of effort is mandatory.
 NONE

This attribute indicates that there is no level of effort.
