Type Definitions
Data Type DiversityConstraints

This data type provides the diversity information for one Call and its supporting connections using the following diversity related parameters:
- coroutingLevelOfEffort
- nodeDiversityLevelOfEffort
- linkDiversityLevelOfEffort
- nodeSRGType
- linkSRGType.

The diversity and co-routing parameter set includes a route group label for each Connection supporting a Call.
Please see SD1-45_ASONControlPlaneManagement-Primer for further detail on diversity use.

Attributes
LevelOfEffort  coroutingLevelOfEffort

This attribute represents (or defines) whether the top-level Connections / Subnetwork Connections in a Route Group are (or shall be) co-routed or not.
This applies within all Route Groups of the Call.
If the value is "None" then there is no co-routing requirement in any of the Route Group.
LevelOfEffort  linkDiversityLevelOfEffort

This attribute indicates the degree to which the Connections in any Route Group are required to be link diverse from the Connections in all other Route Groups of the Call.
If the value is "None" there is no link diversity requirement for the call.
LevelOfEffort  nodeDiversityLevelOfEffort

This attribute indicates the degree to which the Connections in any Route Group are required to be node diverse from the Connections in all other Route Groups of the Call.
If the value is "None" there is no node diversity requirement for the call.
String  linkSrgType

This attribute represents (or defines) the Shared Risk Group Type used (or to be used) for link diversity.

Note: If the "Link Diversity Level of Effort" attribute specifies link diversity (i.e., Best Effort or Mandatory), but no value is specified for this attribute, then basic link diversity is followed, i.e., the Call is routed by using separate Links, and no other link-related risk types, (e.g., duct, bridge, etc.) will be used.

See attached supporting document SD1-45_ASONControlPlaneManagement-Primer.pdf for further detail on diversity use.
String  nodeSrgType

This attribute represents (or defines) the Shared Risk Group Type used (or to be used) for node diversity.

Note: If the "Node Diversity Level of Effort" attribute specifies node diversity (i.e., Best effort or Mandatory), but no value is specified for this attribute, then basic node diversity is followed, i.e., the call is routed by using separate nodes, and no other node-related risk types (e.g., building, floor, etc.) will be used.
