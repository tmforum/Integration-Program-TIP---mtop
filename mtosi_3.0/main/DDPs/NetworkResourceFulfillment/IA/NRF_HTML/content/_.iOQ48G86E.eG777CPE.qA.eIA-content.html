Enumeration RoutingConstraintEffort

This enumeration defines the degree to which the target OS must apply the routing constraints to the route of a Connection.

Enumeration Literals
 BEST_EFFORT

This value indicates that the route of the Connection should try to match the routing constraints however it does not have to match exactly.
 EXACT_MATCH

This value indicates that the route of the Connection must meet the routing constraints exactly.
