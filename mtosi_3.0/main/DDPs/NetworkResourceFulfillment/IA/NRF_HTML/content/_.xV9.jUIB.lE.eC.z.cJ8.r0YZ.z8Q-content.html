Type Definitions
Data Type EquipmentInventory

This data type contains the data structure of the Equipment as defined in the inventory layout.

Attributes
String  eqNm

This attribute contains the name of the Equipment.
«mTOPClass» Equipment  eqAttrs

This attribute contains the specific attributes of the Equipment.
 supportedPtpRefList

This attribute contains the list of Physical Termination Point (PTP) and Floating Termination Point (FTP) names associated with this Equipment.
The PTPs that are returned are those that share their physical layer with the primary equipment (i.e. that represent a port on the equipment or are connected by a fibre, wire, etc.).
The FTPs that are returned are those which are implemented by the physical circuitry supported by the equipment and will include FTPs that are named from the specified equipment (the FTPs will depend upon the presence of that equipment for their most basic operation). When there is equipment protection, this list reports PTPs/FTPs for the primary equipment only. That is, when called on a protecting equipment (as opposed to the primary equipment), this list returns an empty list, irrespective of the current switch status.
 supportedEquipmentRefList

This attribute contains the list of Equipment names supported by this Equipment; e.g., a power pack might support many circuit packs.
 supportingEquipmentRefList

This attribute contains the list of Equipment names that support this Equipment; e.g., the power pack and synchronization card that support this Equipment.
DataQualityIndicator  qualityIndicator

This attribute contains the quality indicator used to report exception information on Equipment data retrieval.
