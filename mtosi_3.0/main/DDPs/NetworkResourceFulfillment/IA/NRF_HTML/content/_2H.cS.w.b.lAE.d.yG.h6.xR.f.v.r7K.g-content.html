Data Type FlowDomainFragmentRoute

This data type represent the route of a Flow Domain Fragment (FDFr).
Traceability: R_TMF518_NRF_I_0057.

Attributes

This attribute represents the partially ordered list of Matrix Flow Domain Fragments (MFDFrs) that constitute the FDFr route.

This attribute represents a summary state of the actual states of the MFDFrs associated to the route.
