Data Type DiversityInfo

This data type provides the information on the diversity that was partially achieved or the diversity violations produced during Call restoration in failure events.

Attributes
String  srgType

This attribute contains the type of Shared Risk Group (SRG) that caused a diversity violation or led to only partial diversity.
SharedResource  sharedResourceList

This attribute contains the optional list of resources causing a diversity violation or where diversity is only partially achieved.
