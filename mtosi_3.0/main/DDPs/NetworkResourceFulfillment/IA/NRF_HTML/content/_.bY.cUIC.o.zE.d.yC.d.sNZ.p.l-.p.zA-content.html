Class OperationsSystem

This object class represents the Operations System (OS) itself (an EMS, NMS or SMS).
In the context of the MTOSI product, there are top-level OSs which are attached to the CCV and subordinate OSs which are known to a given top-level OS but not attached to the CCV.
Traceability: R_TMF518_NRF_I_0001.
