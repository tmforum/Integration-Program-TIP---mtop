Enumeration CallState

This enumeration defines the state values for a Call object.
The call state has dependency upon the state of the associated connections. This dependency is related to call type and policy.
See SD1-35_StateDiagrams for details on valid transitions.

Note: Values 0, 2, 4, 7, and 8 may not be applicable in a non Control Plane environment.

Traceability: R_TMF518_NRF_I_0087.

Enumeration Literals
 ESTABLISHED_IN_SERVICE

This value indicates that all the supporting Connection objects of the Call object have been successfully created.
 ESTABLISHED_IN_SERVICE_DEGRADED

This value indicates that the number of failed Connection objects has reached or exceeded the degraded threshold AND the severely degraded threshold has not been reached or exceeded for the Call object.
 ESTABLISHED_IN_SERVICE_DEGRADED_SEARCHING

This value indicates that at least one supporting Connection object has failed (in the case of LCAS support) AND the severely degraded threshold has not been reached or exceeded for the Call object.
The failed Connections are being restored.
 ESTABLISHED_IN_SERVICE_SEARCHING

This value indicates that the Call object has been modified through the addition of Connection objects and that not all the new Connection objects have been successfully created.
 ESTABLISHED_IN_SERVICE_SEVERELY_DEGRADED

The value of failed supporting Connections has reached or exceeded the severely degraded threshold (in case of LCAS support).
 ESTABLISHED_IN_SERVICE_SEVERELY_DEGRADED_SEARCHING

This value indicates that the number of failed supporting Connection objects has reached or exceeded the severely degraded threshold (in case of LCAS support) for the Call object.
The failed Connections are being restored.
 ESTABLISHED_OUT_OF_SERVICE

This value indicates that all the Connection objects in the Call object have failed and they are NOT being restored.
 ESTABLISHED_OUT_OF_SERVICE_SEARCHING

This value indicates that all the Connection objects in the Call object have failed and they are being restored.
 IN_PROGRESS

This value indicates that the supporting Connection objects of the Call object are currently being created.
