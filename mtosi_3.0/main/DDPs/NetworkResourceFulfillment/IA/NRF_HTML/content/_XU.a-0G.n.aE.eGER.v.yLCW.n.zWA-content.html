Type Definitions
Data Type CallEndType

This data type defines the various ends of a Call or Connection. It has the following fields:
- SNPP
- SNP
- CPTP/FTP/CTP/PTP
- TNA Name (which identifies either a TNA or a Group-TNA Name).

Attributes
String  snppId

This attribute contains the identifier of the SNPP ending a Call/Connection.
String  snpId

This attribute contains the identifier of the SNP ending a Call/Connection.
 tpRef

This attribute contains the name of the TP ending a Call/Connection. One of the end TPs of the Call can be an "off-network" TP (i.e., outside the target OS domain and therefore addressed by a remote address). If this happens in a Control Plane environment, an abstract name may be used that could be equated to the TNA.
See SD1-25 Object Naming for further detail.
String  tnaRefOrGroupTnaRef

This attribute contains either a TNA or a Group-TNA Name for a Call/Connection end.
It is possible to specify here any depth of substructure (e.g. logical port identifier index).
