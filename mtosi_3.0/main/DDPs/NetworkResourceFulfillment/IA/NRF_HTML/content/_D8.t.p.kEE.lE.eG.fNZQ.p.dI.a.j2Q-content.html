Type Definitions
Data Type CallParameterProfile

This data type includes the parameters of the Call. It includes conditions and qualifiers to support a class of service for the Call (e.g., qualifiers that describe LCAS based services).

Attributes
Integer  degradedThreshold

Indicates the threshold (number of working VCAT legs) that qualifies the Call as degraded.
Integer  severlyDegradedThreshold

Indicates the threshold (number of working VCAT legs) that qualifies the Call as severely degraded.
String  classOfService

Indicates the "Class of Service" of the Call.
This parameter applies only to an established Call and shall be ignored during Call creation in a VCAT environment.
 classOfServiceParameters

Contains the "Class of Service" specific parameters.
They are only relevant to an established Call and shall be ignored during Call creation in a VCAT environment.
