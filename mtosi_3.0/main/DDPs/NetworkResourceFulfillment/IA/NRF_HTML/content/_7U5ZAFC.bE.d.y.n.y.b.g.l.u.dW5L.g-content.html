Class RouteChangeNotification

This object contains the information that has to be notified to the NMS when a SNC is rerouted.

Attributes
RouteChangeState  routeChangeState

This attribute identifies the current state of the reroute operation.
The route change goes through different stages during a reroute of a SubnetworkConnection. These stages are a part of this notification.
Possible values are:
- "RerouteStarted"
- "RerouteCompleted"
- "RerouteFailed"
- "RouteActivated", raised when the route is successfully activated (with reference to the route actualState).
- "RouteDeactivated", raised when the route is successfully deactivated (with reference to the route actualState).
