Enumeration DiversitySynthesis

This enumeration defines the values used to identify the type of of diversity violations that have occurred.

Enumeration Literals
 LINK_AND_NODE_VIOLATIONS

This value indicates that the Call object has link and node related diversity violations.
 LINK_VIOLATIONS

This value indicates that the Call object has link related diversity violations.
 NO_VIOLATIONS

This value indicates that the Call object has no diversity violations.
 NODE_VIOLATIONS

This value indicates that the Call object has node related diversity violations.
 NOT_APPLICABLE

This value indicates diversity violations are not relevant for the Call object.
 UNKNOWN

This value indicates that it is not known if diversity violations are relevant for the Call object.
 VIOLATIONS

This value indicates that the Call object has diversity vioaltions, but it is not known if they are node or link related.
