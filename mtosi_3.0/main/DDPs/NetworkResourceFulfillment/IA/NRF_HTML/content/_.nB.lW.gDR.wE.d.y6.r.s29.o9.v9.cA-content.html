Class TerminationPointPool

This object class represents a set of Termination Points or Group Termination Points that are grouped. Properties that are common for all the TPs/GTPs in the TPPool are identified in the multiple associated commonTransmissionParameters.
Traceability: R_TMF518_NRF_I_0014 and R_TMF518_NRF_I_0015.

Attributes
Integer  numberOfMembers

This attribute identifies the total number of TPs or GTPs currently contained in the TPPool.
Integer  numberOfIdleMembers

This attribute identifies the number of currently contained idle TPs or GTPs in the TPPool.
 commonTransmissionParameterList

This attribute contains the common layers and Transmission Parameters associated with the TPs grouped by the associated TpPool (or contained by the GTPs that are grouped by the associated TPPool).
Refer to attached supporting document SD1-16_LayeredParameters for details of the currently defined Transmission Parameters.
String  descriptionOfUse

This attribute contains a description of the specific use of the TPPool.
The description of use of an ATM VP TPPool would be "VirtualUNI", if all its VP CTPs are contained in the same real user network interface (UNI) (i.e., a tributary ATM NI CTP) that is partitioned into virtual UNIs (i.e., VP TPPools).
«mTOPClass» GroupTerminationPoint  containedGtpMemberList

This relationship contains the list of Group Termination Point (GTP) names that are contained by the TPPool.
«mTOPClass» TerminationPoint  containedTpMemberList

This relationship contains the list of Termination Point (TP) names that are contained by the TPPool.
