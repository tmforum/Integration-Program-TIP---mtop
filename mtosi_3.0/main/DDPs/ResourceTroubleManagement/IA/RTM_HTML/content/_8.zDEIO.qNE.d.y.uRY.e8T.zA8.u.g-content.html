Type Definitions
Data Type AlarmId

This data type is used as a unique identifier of an Alarm.

Attributes
 objectName

This attribute represents the name of the entity that gave rise to the Alarm.
See supporting document SD2-7_Object_Naming for further details on the name structure.
 layerRate

This attribute identifies the layerRate of the object raising the Alarm. For objects where the layerRate is not applicable, such as EMS/OS, the value is set to LR_Not_Applicable. LayerRate is applicable in Alarms raised by objects such as PTPs.
String  probableCause

This attribute identifies the type of Alarm raised against the object.
See supporting document SD1-33_ProbableCauses for a description of the probable cause strings.
String  probableCauseQualifier

This attribute is used as the final component of the unique identification of the Alarm and is left blank where the objectName, layerRate and probableCause alone provide a unique identification of the alarm.
