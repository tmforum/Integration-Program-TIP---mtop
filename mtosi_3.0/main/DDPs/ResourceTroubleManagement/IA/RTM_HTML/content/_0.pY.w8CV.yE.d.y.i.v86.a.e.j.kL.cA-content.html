 getAlarmSeverityAssignmentProfileByResource 
Public  getAlarmSeverityAssignmentProfileByResource(, Integer,  )

This operation retrieves the ASAP(s) which are assigned to the addressed resource. The ASAP list can have zero or more elements, as all resources but TPs can refer to at most one ASAP. TPs can refer to more ASAPs, at most as many as the number of encapsulated layer rates.

Traceability: R_TMF518_RTM_II_0010.

Parameters:
asapList [*]This parameter contains the list of ASAPs assigned to the addressed resource with their complete information or empty list.
Integer layerRateList [*]This parameter contains the list of resource layer rates which assigned ASAPs are to be retrieved. If an empty list is specified, then all ASAPs assigned to the addressed resource will be replied. The list shall also be empty if the addressed resource is not a Termination Point.
resourceRef [1]This parameter contains the name of the resource to which the ASAPs that shall be retrieved are assigned to.
Returns:
void [1]
