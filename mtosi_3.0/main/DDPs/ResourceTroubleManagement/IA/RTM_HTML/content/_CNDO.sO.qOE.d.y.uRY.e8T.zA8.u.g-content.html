Type Definitions
Data Type TcaId

This data type is used as a unique identifier of a Threshold Crossing Alert.

Attributes
 objectName

This attribute represents the name of the entity that gave rise to the TCA.
See supporting document SD2-7_Object_Naming for further details on the name structure.
 layerRate

This attribute identifies the layerRate of the object raising the TCA. For objects where the layerRate is not applicable, such as EMS/OS, the value is set to LR_Not_Applicable. LayerRate is applicable in Alarms raised by objects such as PTPs.
String  pmParameterName

This attribute holds the name of Performance Measurements. It has been defined as a string to accommodate proprietary extensions.
See attached SD1-28_PerformanceParameters supporting document for the list of defined Performance Measurements.
 pmLocation

This attribute identifies the location of the PM measurent.
See attached supporting document SD1-19_LocationIdentification.
 granularity

This attribute identifies the granularity (count period of the measure) for which the Threshold Crossing Alert was raised.
