Type Definitions
Data Type ProtectionGroupSetData

This data type provides the Protection Group attributes which can be set at creation and modification time.

Traceability: R_TMF518_RTM_II_0071, R_TMF518_RTM_II_0073.

Attributes
 reversionMode

This attribute identifies whether the protection scheme shall be revertive or not.
 terminationPointRefList

This attribute identifies the complete intended list of Protection Group TP names. It is a partially ordered list as in the Protection Group structure, i.e. workers names are first and protecting TP names follow in order of priority. Thus each TP has a well defined position in the list according to its role and priority.
In normal operation the list shall contain a number of TPs specific to the protection type however under reconfiguration scenarios there may be more or less TPs than normal for a short period. However, the terminationPointList should never be empty as Protection Groups without TP members are not permitted.
 parameterList

This attribute contains a name value list for the parameters of the Protection Group to be created or modified.
See attached SD1-29_PGPParameters supporting document for details (values and descriptions) on Protection Group parameters.
String  apsProtocolType

This attribute defines the APS protocol on which the switching function shall be based on.

Possible values:
- "G.783" in case of K1/K2 based APS
- "Legacy" in case of proprietary APS protocol, as for radio equipment.
 asapRef

This attribute defines the assignment of an Alarm Severity Assignement Profile (ASAP) to the Protection Group.
