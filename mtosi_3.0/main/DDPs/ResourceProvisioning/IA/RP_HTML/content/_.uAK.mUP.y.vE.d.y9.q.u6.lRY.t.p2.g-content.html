Type Definitions
Data Type RouteNameAndAdminState

This data type contains a Route identifier and their administrative state.

Attributes
String  routeId

This attribute contains the unique identifier of a Route within the Subnetwork Connection.
 administrativeState

This attribute specifies the administrativeState of the Route.
Note: There is not any relationship with the actualState attribute of the Route.
It can have the following values:
- "locked", the Route is not allowed to be active.
- "unlocked", the Route is allowed to be active.
 additionalInfo

This parameter allows the communication of additional information which is not explicitly modelled.
