Enumeration GroupTerminationPointEffort

This enumeration defines a statement of the requirement of the list of CTPs for a new GTP.
If SAME is specified then the target OS must create the GTP with the exact same list of CTPs as provided with the GTP creation request. Further, if the requesting OS uses SAME, it is assumed that the target OS will not modify the CTP list comprising the GTP at some later point (unless requested via the modifyGroupTerminetionPoint operation).
If WHATEVER is specified then the target OS may comply with the total bandwidth requirement by using a different set of CTPs.
