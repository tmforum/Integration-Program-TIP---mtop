Interface MultiLayerSnppLinkControlService

Operations
 assignSignallingController

This operation assigns a Multi Layer SNPP Link to a Signaling Controller. For this operation to be successful, the link must not be associated with a Signalling Controller. In case the link is already associated to a Signalling Controller, the link needs to be disabled and then dissociated from the Signalling Controller.

Traceability: R_TMF518_RP_II_0123, R_TMF518_RP_II_0125, UC_TMF518_RP_0073.
 deassignSignallingController

This operation de-assigns a Multi Layer SNPP Link from a Signaling Controller. After the successful completion of this operation the signallingControllerIdentifier attribute of the Multi Layer SNPP Link is set to an empty string.

Traceability: R_TMF518_RP_II_0124, UC_TMF518_RP_0078.
 enableSignalling

This operation enables the signalling on a Multi Layer SNPP Link.

Traceability: R_TMF518_RP_II_0126, UC_TMF518_RP_0076.
 disableSignalling

This operation disables the signalling on a Multi Layer SNPP Link.

Traceability: R_TMF518_RP_II_0127, UC_TMF518_RP_0077.
 setSignallingProtocolAndParameters

This operation sets the signalling protocol and parameters to be used by the signaling interface associated with the addressed Multi Layer SNPP Link.

Traceability: R_TMF518_RP_II_0125, UC_TMF518_RP_0074.
 modifySignallingProtocolParameters

This operation modifies the signalling parameters to be used on the signalling interface associated with the addressed Multi Layer SNPP Link. The operation can be performed when signalling is enabled or disabled.

Traceability: R_TMF518_RP_II_0125, UC_TMF518_RP_0075.
 setMlsnppLinkTnaName

This operation sets TNA Name(s) to one or more components of the Multi Layer SNPP Link. The operation also allows the un-setting of TNA Name(s) by providing an empty value as the TNA Name(s). The TNA Name(s) set by this operation will overwrite existing TNA Name(s).

Traceability: R_TMF518_RP_II_0128, UC_TMF518_RP_0080.

 setMlsnppLinkTnaName

mlsnppLink [1]This parameter returns the Multi Layer SNPP Link with the new values assigned.
mlsnppLinkRef [1]This parameter provides the unique identifier of the Multi Layer SNPP Link.

SnpTnaData snpTnaDataList [*]This parameter provides the list of SNP Identifiers and the corresponding TNA Names to be set.
String zTnaGroupName [1]This parameter provides the TNA Group Name to be assigned to the z-End of the link (i.e. the zTnaGroupName attribute).
String zTnaName [1]This parameter provides the TNA Name to be assigned to the z-End of the link (i.e. the zTnaName attribute).

void [1]
