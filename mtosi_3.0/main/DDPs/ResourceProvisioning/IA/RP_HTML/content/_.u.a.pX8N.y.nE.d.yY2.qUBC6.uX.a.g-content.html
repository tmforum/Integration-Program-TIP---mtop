Interface TopologicalLinkControlService

Operations
 createTopologicalLink

This operation creates a Topological Link; top-level as well as intra-MLSN.
The behaviour of the operation should be the same as if a target OS user were to attempt to create a Topological Link (or equivalent operation) at the target OS user interface.
Therefore, the specific conditions under which the creation is rejected are left to the target OS implementation.
Note: It is at the discretion of the target OS to delete Topological Links created by the requesting OS, as the owner of the Topological Link is the target OS, not the requesting OS.

Traceability: R_TMF518_RP_II_0100, R_TMF518_RP_II_0101, UC_TMF518_RP_0045.
