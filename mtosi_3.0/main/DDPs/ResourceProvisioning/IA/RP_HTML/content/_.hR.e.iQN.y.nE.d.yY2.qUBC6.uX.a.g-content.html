Interface EquipmentProvisioningService

Operations
 provisionEquipment

This operation allows the requesting OS to provision an Equipment in an Equipment Holder in a Managed Element. The successful result of this operation is the creation or update of the Equipment object and the creation of all its related objects such as Termination Points. The Equipment object may already exist, but the Equipment Holder must not have an expected Equipment already provisioned.
See attached supporting document SD2-20_EquipmentModel.

Traceability: R_TMF518_RP_II_0034, R_TMF518_RP_II_0035, UC_TMF518_RP_0013.
 unprovisionEquipment

This operation allows the requesting OS to permanently unprovision an Equipment from a Managed Element. The successful result of this operation is the deletion of the equipment object and the deletion of all its related objects such as Termination Points. The target OS should attempt to set the Equipment's service state to OUT_OF_SERVICE_BY_MAINTENANCE before it unprovisions it.
Traceability: R_TMF518_RP_II_0036, UC_TMF518_RP_0012.
