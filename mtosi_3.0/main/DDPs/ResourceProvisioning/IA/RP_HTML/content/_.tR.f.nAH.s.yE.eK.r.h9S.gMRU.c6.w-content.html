 setMlsnppTnaName

 setMlsnppTnaName 
Public  setMlsnppTnaName(, , SnppTnaPair, SnpTnaData, String, String )

This operation sets TNA Name(s) to one or more components of the Multi Layer SNPP. The operation also allows the un-setting of TNA Name(s) by providing an empty value as the TNA Name(s). The TNA Name(s) set by this operation will overwrite existing TNA Name(s).

Traceability: R_TMF518_RP_II_0129, UC_TMF518_RP_0079.

Parameters:
mlsnpp [1]This parameter returns the Multi Layer SNPP with the new values assigned.
mlsnppRef [1]This parameter provides the unique identifier of the Multi Layer SNPP.
SnppTnaPair snppTnaPairList [*]This parameter provides the list of SNPP Identifiers and the corresponding TNA Names to be set.
SnpTnaData snpTnaDataList [*]This parameter provides the list of SNP Identifiers and the corresponding TNA Names to be set.
String tnaGroupName [1]This parameter provides the TNA Group Name to be assigned to the Multi Layer SNPP.
String tnaName [1]This parameter provides the TNA Name to be assigned to the Multi Layer SNPP.

void [1]
