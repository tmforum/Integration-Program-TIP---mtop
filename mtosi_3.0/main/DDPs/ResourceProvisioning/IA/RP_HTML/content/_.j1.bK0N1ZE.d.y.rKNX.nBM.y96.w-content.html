Type Definitions
Data Type FloatingTerminationPointCreateData

This data type represents the parameters that may be provided when creating a Floating Termination Point.

Traceability: R_TMF518_RP_II_0091.

Attributes
 equipmentRef

This attribute defines the identifier of the Equipment hosting the new Floating Termination Point in the Network Element.
 ingressTmdRef

This attribute identifies the required ingress Transmission Descriptor for the new Floating Termination Point.
 egressTmdRef

This attribute identifies the required egress Transmission Descriptor for the new Floating Termination Point.
 direction

This attribute identifies the required direction of the Floating Termination Point.
 transmissionParameterList

This attribute contains the layer rate on which connectivity is established and a related set of layered parameters.
Refer to supporting document SD1-16_LayeredParameters for details of the currently defined layered parameters.
 tpMappingMode

Within the façade definition, the CTP/FTP can act as an aggregate of associated G.805 TCPs, G.805 Termination Functions and G.805 CPs at one or more LayerRates. The CTP is contained within a PTP or FTP.
The tpMappingMode attribute indicates and controls the connection of the named CP at a specified LayerRate to the dedicated G.805 TCP and associated G.805 Termination Function at the same LayerRate within the CTP/FTP.
