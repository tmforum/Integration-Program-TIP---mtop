Interface SoftwareAndDataControlService

This interface performs software management related functions of the target OS.

Operations
 abortManagedElementBackup

This operation will abort a backup of ME configuration data that is pending or in progress. If there is no backup of configuration data of the ME pending or in progress an exception will be thrown.

Traceability: R_TMF518_RP_II_0075.
 backupManagedElement

This operation initiates a backup request on the specified Managed Element. The target OS sends the results of the backup operation for the Managed Element as Notifications to the Notification Service.
Traceability: R_TMF518_RP_II_0073.
 getBackupList

This operation is used to determine what backups exist in the target OS for a specified list of Managed Elements. It returns the list of available backups for the specified Managed Elements. If the ME list is empty, all database backups will be returned (note that only the names are returned not the backups themselves).

Traceability: R_TMF518_RP_II_0076.
 getManagedElementBackupStatus

This operation retrieves t the current status of the database backup of an individual Managed Element.

Traceability: R_TMF518_RP_II_0074.

 getBackupList 
Public  getBackupList(BackupIdentifier,  )

This operation is used to determine what backups exist in the target OS for a specified list of Managed Elements. It returns the list of available backups for the specified Managed Elements. If the ME list is empty, all database backups will be returned (note that only the names are returned not the backups themselves).
