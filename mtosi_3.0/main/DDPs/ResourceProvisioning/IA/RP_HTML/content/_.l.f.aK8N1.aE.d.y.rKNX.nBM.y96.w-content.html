Type Definitions
Data Type TopologicalLinkCreateData

This data type represents the parameters that may be provided when creating a Topological Link.

Traceability: R_TMF518_RP_II_0101.

Attributes
 direction

This attribute defines the direction of the Topological Link.
Note: A Topological Link can be unidirectional even if both of its ends are bidirectional TPs.
 layerRate

This attribute identifies the layer at which the Topological Link is to be created.
Boolean  isReportingAlarms

This attribute indicates whether alarm reporting for the Topological Link is administratively activated or de-activated.
True = alarm reporting is activated; False = alarm reporting is de-activated.
 aEndTpRef

This attribute contains the requested A end Termination Point of the new Topological Link.
The Termination Point name must be explicit; i.e., a generic endpoint specification may not be used in this case.
 zEndTpRef

This attribute contains the requested Z end Termination Point of the new Topological Link.
The Termination Point name must be explicit; i.e., a generic endpoint specification may not be used in this case.
 asapRef

This attribute indicates the required assignment of an Alarm Severity Assignement Profile (ASAP) to the Topological Link.
