The purpose of this folder is to show examples of XML documents generated from the mTOP XSD.
Content of this folder includes:
- me.xml         - example of an ME mTOP network resource XML element with mTOP and vendor extensions
- MtopExt.xsd    - example of mTOP schema extensions for Fulfillment DDP definitions (ME)
- Vendor1Ext.xsd - example of vendor1 schema extensions (ME)
- Vendor2Ext.xsd - example of vendor2 schema extensions (ME)
