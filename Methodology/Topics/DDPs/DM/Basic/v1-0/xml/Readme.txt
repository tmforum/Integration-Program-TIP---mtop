The purpose of this folder is to show examples of XML documents generated from the mTOP XSD.
Content of this folder includes:
- comRscInfo.xml        - example of a mTOP CommonResourceInfo XML element with extensions (mTOP & vendor)
- MtopCommonExt.xsd     - example of mTOP schema extensions for Basic DDP definitions (CommonRscInfo)
- Vendor1CommonExt.xsd  - example of vendor1 schema extensions (CommonRscInfo)
