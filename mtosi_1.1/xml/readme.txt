The purpose of this xml folder is to capture sample MTOSI XML messages

inventory - include sample getInventory messages (request, response, and fault)
notifications - include sample notification (notify) messages of all supported MTOSI events
vendorSchema - include sample XSD of vendor XML Schema used in XML samples

Notes:
- All XML sample documents are generated with dummy data
- All XML sample documents have xsi:schemaLocation attributes defined with local XSD files to allow for XML Schema validation
