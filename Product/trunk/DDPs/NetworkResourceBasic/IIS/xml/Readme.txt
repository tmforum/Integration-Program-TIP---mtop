The purpose of this folder is to capture examples of XML documents generated from the NRB schema (XSD)
Content of this folder includes:

- comRscInfo.xml: example of a CommonRscInfo element with minor & vendor extensions
- Cri_MinorExt.xsd: example of two minor extensions of the cri/v1 namespace (normally defined into xsd/ext/CommonResourceInfoExt.xsd)
- CriInstance.xsd: Define element for CommonRscInfo
- Vendor1CommonExt.xsd : example of vendor1 schema extensions (CommonRscInfo)
